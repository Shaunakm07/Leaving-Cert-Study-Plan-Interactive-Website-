# Study Plan 2026

An interactive 4-week study plan covering 7 subjects and 164 topics.

## Subjects
- English (every day)
- Physics (every day, max 1 chapter per day)
- Biology
- Chemistry
- Maths
- Applied Maths
- Computer Science

## Features
- ✅ Mark topics as done by clicking them
- ⏰ Add custom time slots to each topic per day
- 🔍 Filter by subject
- 📅 Navigate by week or view all
- 🔁 Repeat topics highlighted with ↩
- 💾 Progress saved automatically in localStorage

## Usage

Just open `index.html` in any browser — no build step, no dependencies, no server needed.

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your plan will be live at `https://yourusername.github.io/repo-name`
